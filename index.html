How to set up Visual Studio?

You can easily set up visual studio using this link: https://code.visualstudio.com/Download. Just go here and click the box that has your type of computer on it. Then it will download the app and just open it and get started!

What is the console screen?

Console is itself a program which your operating system has. This is a low level command line interface used for communication between user,your application and system.
